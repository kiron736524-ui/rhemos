# 高度、结构、跨度与顶部 Truss

固定事实规则，用于约束方案不出现不可落地结构，并减少无效追问。**跨度安全是后台硬规则，不交给用户偏好。**

## 一、高度体系（最易混淆，务必厘清）

- **"展台高度一般 6m"通常指总高度约 6m**，包含 Truss、吊挂标识、顶部灯架或门头最高点；**不要把 6m 误写成所有板墙和房间都做到 6m**。
- **主体板墙高度**（不含 Truss）按地区：国内常用 ~**4.4m**，国外常用 ~**4.0m**。用户未说国家时，中文国内项目默认 4.4m、英文/海外项目默认 4.0m，并在确认时标注为默认。
- **储物间/会议室/封闭洽谈室外墙**：贴主墙、承接 LED 墙、形成主立面或与储物间并列时，优先与 4.0-4.4m 主体板墙**齐平**，形成连续体量，而不是做成 2.5m 矮盒子。
- **2.5-2.8m** 只适合最低隐私/门高尺度或半封闭隔断高度。

## 二、跨度安全（硬规则，不让用户偏好覆盖）

- 普通**木结构**无中间支撑跨度 **≤ 约 6m**；普通**立柱桁架**无中间支撑跨度 **≤ 约 9m**。超过必须增加支撑或改结构。
- 出现大门头、大屏、大跨桁架、悬浮体块时自动套用此规则；方案给用户的是"加支撑 / 缩短跨度 / 改分段结构 / 降低顶部体量"，而不是问"能不能超跨度"。

## 三、地面、墙体、屏幕与后场

- 地面：低预算优先地毯/简单地台；中高预算常用 ~10cm 地台配地板/地毯/局部发光边，便于走线和抬升层次。
- Truss/桁架只有两类支撑逻辑：**场馆吊点吊装** 或 **落地立柱支撑**。**没有吊点不能写成悬浮桁架。** 立柱应靠边或结合墙体，避免落在主入口和产品操作区。
- LED 大屏常嵌入木板墙，背后需检修/布线/储物空间，预留约 60cm-1m，**~80cm 是常用安全值**。
- 电视屏可壁挂/嵌墙/结合立柱；多台电视应确认集中展示还是分散到业务分区。
- 储物间优先藏在主视觉墙后、洽谈室后或边角，不占主通道视觉中心。
- 灯具需真实来源：金卤灯、轨道射灯、墙面灯、灯带、筒灯、软膜灯箱都要挂靠在顶部/墙体/柜体/结构边缘。

## 四、顶部与 Truss 中部造型（把"有没有架子"提升为天际线设计）

### 核心原则
- 每个展台都要处理顶部策略：无顶部 / 局部门头 / 落地 Truss / 吊装 Truss / 平顶吊顶 / 张拉天幕 / 吊挂标识。
- **Truss 不能只是一圈裸露铁架**，除非用户明确要工业风/设备感/演出感或选"开放空心"。多数品牌展台需要中部做与场景适配的应景造型（边框、灯带、吊牌、软膜、网格、环形、几何）。
- 顶部至少三层：外围轮廓（远处先看到）→ 中部造型（站内抬头看到）→ 灯光/品牌层（解释品牌、引导焦点）。
- 顶部中部造型必须回应地面布局：中心岛↔环形/方形吊挂；线性动线↔条形灯阵；分区展示↔模块化网格；商务洽谈↔柔和天幕。
- 顶部不压迫主通道、不遮主视觉/LED、不让产品区变暗；中部造型留视觉呼吸。

### 必问项（进完整方案前确认或显式默认）
1. 顶部结构方式：无顶/局部门头/落地 Truss/吊装 Truss/平顶吊顶。
2. Truss/顶部中部造型：开放空心/环形椭圆/方形矩形灯框/条形网格灯阵/张拉软膜网布天幕/几何模块悬浮盒子。**一般要确认"是否做中部造型、做什么"，不能默认空着。**
3. 品牌识别位置：外围吊牌/四面吊旗/中央品牌环/门头品牌/主视觉墙品牌。
4. 顶部与产品/动线关系：焦点压在主展品/接待入口/互动区/洽谈区/中心公共区。

### 两轮提问法
**第一轮·顶部策略**（带空间后果）：
- 无顶部/强化墙体门头：预算低、空间开阔，但远距离识别弱。
- 局部门头/局部顶框：成本可控，入口和主视觉更强。
- 落地 Truss：可挂灯和品牌、结构感强，但立柱必须靠边。
- 吊装 Truss：地面最干净、顶部完整，但依赖场馆吊点、限高、审批。

**第二轮·中部造型**（带美感后果）：
- 开放空心/裸架：最省、通透、不抢产品，但记忆点弱；只适合低预算/工业设备感/明确不要造型。
- 环形/椭圆吊挂：适合四面开、三面开、中心岛，360° 识别强（科技/汽车/消费电子/高端品牌）。
- 方形/矩形灯框：秩序感强，配长方形展位、直线动线（科技/电子/工业）。
- 条形/网格灯阵：适合线性动线、多品类、科技感，引导参观方向。
- 张拉软膜/网布天幕：氛围柔和、包裹感强（生活方式/文旅/健康/食品/美妆），注意消防喷淋。
- 几何模块/悬浮盒子：记忆点强，适合高预算/新品发布/概念展示，成本与吊挂复杂度高。
- 场景化主题造型：按行业语义定制（能源→流线光带；水产/海洋→波浪软膜；汽车科技→速度线灯阵；医疗健康→柔和环形光带）。必须服务场景，不做无意义装饰。

### 美感规则（按开口与行业）
- 一面开/两面开：顶部优先服务入口、主视觉墙、前场吸引，不做过重中央吊挂。
- 三面开/四面开：顶部优先形成中心识别点（环形/方形/十字灯阵/分散模块优于单一背墙）。
- 小面积：少做满铺 Truss，以局部门头/轻量灯框/软膜小天幕为主。
- 大面积：顶部要有节奏，避免一大片空顶（外围轮廓+中央吊挂+分区灯阵形成层次）。
- 科技/工业：允许外露 Truss、矩形灯框、线性灯阵、机械感节点。
- 高端商务：弱化裸露桁架，用软膜/内发光吊牌/干净金属边框/暖色线性灯隐藏结构感。
- 文化/生活方式：可用织物天幕、弧形吊挂、木色格栅、植物吊挂，但不挡喷淋和吊点安全。

## 五、安全与可落地

- 吊装 Truss、吊牌、吊旗、软膜天幕、吊灯、LED 吊屏都必须有吊点、载荷、审批逻辑；**未知吊点只能作待确认项，不能写成确定方案**。
- 落地 Truss 立柱必须靠边/贴墙/结合储物间或品牌柱，不得落在入口、主通道、演示操作区、大型产品回旋区。
- 张拉软膜、网布天幕、吊挂植物要考虑消防喷淋、透水/透烟性、展馆规则；不默认做完全封闭顶棚。
- 所有顶部灯具/吊牌/屏幕/装饰件都要有明确挂靠对象，**不出现悬浮结构**。

## 六、补充：地面 / 墙体厚度 / 无顶补偿 / 双层

- **地面 4 类**：直接铺地毯 / 地台+地板 / 地台+地毯 / 直接铺地板。地台常用 **5cm 或 10cm**，作用不只是抬高，更是藏走线、形成边界、提升完成度。**有完整电子设备/灯光/屏幕时优先理解为地台或可藏线地面**，不是一层极薄平地板。
- **墙体是空间容器，不是薄竖板**：成熟展台里墙体 = 空间容器 + 品牌载体 + 展示骨架，可形成储物间/洽谈室/会议室/画面墙/灯箱墙/电视墙/LED墙/内嵌橱窗。
- **墙体加厚规则**：普通板墙默认 ~10cm；但墙体很长、要挂重物、要嵌设备、要做造型、要承担展示结构时**必须加厚**。**不要出现"极薄长墙 + 挂重屏 + 大跨度 + 完全无补强"。**
- **无顶部补偿逻辑**：不做 Truss 顶 ≠ 空盒子——要靠墙体造型、门头体量、立面厚度、内嵌展陈、异形轮廓补足空间感（**有顶靠上部、无顶靠立面**）。
- **双层展台**：总高 ~6m（个别 6.5m），首层 ~3m；常有玻璃栏板和完整上下体量关系。不要画成漂浮悬空盒子，也不要层高压得过低。
- **吊装 vs 柱撑（生图理解）**：吊装型 = 上部悬挂逻辑清楚；柱撑型 = 下部支撑逻辑清楚。表现成立感即可，不必画施工细节。
