# 开口、方位、动线与空间预算

固定领域规则。优先用于判断墙体、主视觉面、入口与动线位置；不是风格建议。

## 一、开口类型 → 骨架

- **一面开**：通常三面靠邻展/场馆边界，三面可做墙，唯一开放面朝通道；主视觉面放最深处或正对入口。非正方形时必须确认开放面在**长边还是短边**：长边开放强调宽门面、横向展示；短边开放强调纵深序列、内向参观。
- **两面开**：不能只写"两面开"，必须区分：
  - **相邻两面开**（角位/corner）：两条开放边成 L 形转角，两条邻展边可做 L 型墙；适合转角吸引、斜向主视觉、双入口接待。
  - **相对两面开**（pass-through/穿越）：两条开放边平行，另两侧为平行墙/邻展；适合贯通动线，但**必须防止观众把展台当通道穿过**——接待/产品/品牌墙要布在动线侧边形成停留点。还要确认开放的是两条**长边**（宽向双面展示）还是两条**短边**（深向穿越）。
- **三面开**（半岛）：通常一面靠邻展，必须形成后场主板墙；展示集中在后墙，前区两侧避免大面积封堵。非正方形时必须确认唯一背墙是**长边背墙**（宽幅主视觉/LED/连续品牌墙）还是**短边背墙**（更深的参观序列，主视觉窄但纵深强）。
- **四面开**（岛型）：无邻展强制墙，但不是不能做墙——允许边缘品牌墙、L 型边缘墙、半墙、低矮展柜、悬挂门头；**高墙、储物间、会议室不得居中阻断动线**。四面开仍必须确认主通道方向，否则品牌朝向和首图视角会漂移。

### 骨架默认模板（多数展台适用）
主轴线（观众从主通道看进来的主方向）→ 轴线尽端放主视觉面（展台的"脸"：LED 主屏/品牌墙/核心产品）→ 左右分区（沿主轴两侧展开功能区）→ 前区/入口（接待台/开放展示/引流）。**先骨架后填充：展台是一个建筑，不是一堆家具。**

## 二、方位与长短边规则

- 记录尺寸**必须保留长/短边语义**：6×12m 写成"12m 长边 × 6m 短边"，不要只写 72㎡。
- 一面开、三面开、相对两面开遇长方形展位时，"几面开"不足以生成方案，必须补齐至少一项：背墙/邻展墙靠长边还是短边；开放边靠长边还是短边；哪条开放边是主通道/最大人流方向。

## 三、邻展与通道规则

- 邻展面必须围合或遮挡，不能把背板、储物门、杂物面暴露给邻展或主通道。
- 开放边需保护通透性；高体量、封闭房间、大屏墙应靠边/靠后/贴邻展面，不应压在入口正中。
- 主视觉墙、LED 主屏、品牌墙优先朝向主通道；主通道方向不明时不要写死方案，先确认，或在用户授权"你来判断"时按最大开放边默认并标注。
- 接待台靠主入口/开放转角/主通道侧，不堵主轴线和产品动线。
- 四面开需要品牌识别时，优先用边缘品牌墙、门头、吊挂标识或局部半墙创造展示面，而非中央高墙。

## 四、动线设计：引导，不阻挡

| 展台规模 | 动线模式 |
|---|---|
| 小型（≤36㎡） | 直线穿越型：入口→主视觉→出口 |
| 中型（36-72㎡） | L 型或 U 型：入口→主视觉→侧展示→回前区 |
| 大型（72-150㎡） | 环形型：入口→主视觉→左→后→右→回前区 |
| 超大型（150㎡+） | 多入口分流型：多入口各自引导到不同功能区 |

**动线由开口推导**：一面开→线性；相邻两面开→L 型/转角导入；相对两面开→穿越但可停留；三/四面开→环形或多入口导流。
**动线规则**：主通道到主视觉面之间不能有遮挡（一眼看到主角）；洽谈/VIP 放动线末端或侧面；产品沿动线布置；接待台在入口附近（第一接触点）。动线必须先避开大型设备、封闭洽谈室、Truss 立柱、储物间门。

## 五、产品摆放（必主动引导，不等用户补）

四类组织方式产生完全不同的骨架：**集中主展区**（焦点强，适合单核心产品/新品发布）/ **沿动线分散**（多品类，需更清晰导视分区）/ **入口爆品吸引**（获客强，但不能堵入口，前方留停留回旋）/ **演示体验优先**（必须留操作台、讲解位、围观位、设备维护空间）。

## 六、面积 ↔ 功能区匹配（空间预算）

| 面积范围 | 功能区数量 | 建议重心 |
|---|---|---|
| 9-18㎡ | 2-3 个 | 主展示+接待，最多加一个小洽谈 |
| 18-36㎡ | 3-4 个 | 主展示+产品区+接待+洽谈 |
| 36-72㎡ | 4-5 个 | 完整配置，可有封闭洽谈+储物 |
| 72-150㎡ | 5-7 个 | 全功能+舞台/演示+VIP |
| 150㎡+ | 7+ 个 | 多主题分区，可做双层 |

**空间预算规则**：功能超出面积承载时，不要全挤进去（全做小不如精做几个）；优先保证核心目的对应的功能区面积；洽谈区和储物间最容易缩减；必须让用户做取舍（给可点选项，说明面积代价）。

## 七、功能区标准配置（常识尺度）

| 功能区 | 标准配置 | 尺度参考 |
|---|---|---|
| 品牌主墙/LED | 高墙+嵌入屏/灯箱+品牌标识 | 宽度约占后墙 60-80% |
| 产品展示 | 展柜/展台+射灯+说明牌 | 展柜间距约一臂宽，留人通行 |
| 接待台 | 吧台台面+品牌标识 | 入口显眼处，台面约胸高(90cm) |
| 洽谈区 | 桌椅+半/全封闭隔断+软装 | 4 人约 4-6㎡，6 人约 6-9㎡ |
| 演示/舞台 | 低台+屏幕+站立空间 | 台高约膝高，面积看观众数 |
| 体验互动 | 操作台+产品+引导标识 | 每互动位约 2-3㎡ |
| 储物间 | 封闭空间+门 | 藏在主视觉面后方 |

## 八、柜体与产品匹配

柜体核心目的：**展示展品 + 放资料**。形态可方/圆/梯/台阶/异形定制，但**不能脱离产品逻辑**——先判断"展示什么"，再定柜体高度与形式。柜体也可**内嵌墙体**形成一体化展示（内嵌橱窗/展示柜/屏幕都是成熟做法），不必全独立摆放。
- 小件/电子展品：展柜高 **75-100cm**（视线高度）；手机等小而贵的产品放柜面或玻璃柜，隐含防盗固定逻辑。
- 大件产品（冰箱/洗衣机/摩托/轮椅/设备类）：**直接落地，或只放 10-30cm 小地台**，**绝不放在高柜上**。
