# 展台设计方法论

把"信息收集完毕"转化为"有空间骨架的成熟方案"时，用这个框架组织设计判断。这是基础框架，不替代真正的高级设计师经验（专家技法可随用随补）。

## 一、空间构图：先骨架后填充

**展台是一个建筑，不是一堆家具。** 出方案第一步不是列功能区清单，而是建立空间骨架：
1. **主轴线** — 观众从主通道看进来，视线主方向是什么？决定整个展台的正面感。
2. **主视觉面** — 轴线尽端放什么？这是展台的"脸"（LED 主屏 / 品牌墙 / 核心产品）。
3. **左右分区** — 主视觉面确定后，左右两侧各承担什么功能，沿主轴两侧展开。
4. **前区** — 靠通道的前区是"入口"（接待台 / 开放展示 / 引流装置）。

```
              ┌─────────────────────────┐
              │      主视觉面/背景墙      │  ← 轴线尽端
              ├────────────┬────────────┤
              │   左功能区   │   右功能区   │  ← 主体分区
              │ (封闭/半封闭) │  (开放展示)  │
              ├────────────┴────────────┤
              │        前区/入口区         │  ← 靠近通道
              └─────────────────────────┘
                        ↑ 主通道（观众来向）
```
（开口方式如何改变骨架、动线、面积↔功能区匹配，见 `space-opening-circulation`。）

## 二、主次层级：一个展台只能有一个主角

视觉重心唯一——观众 5 秒内必须知道"这个展台想让我看什么"。
- **主视觉（注意力 60%+）**：大型 LED / 品牌主形象墙 / 核心产品舞台。必须在主轴线上、最显眼处，所有灯光色彩体量向它收拢。
- **次要展示（20-30%）**：产品展示 / 体验互动。位于主视觉两侧或前方，材质色彩呼应主视觉，但体量明显更小。
- **功能支撑（<10%）**：接待台 / 洽谈区 / 储物间。不抢主视觉，用材质和高度差暗示"后勤区域"。

常见错误：❌ 每个功能区一样重要（观众不知看哪）；❌ 洽谈区比展示区更显眼（主次颠倒）；❌ 小展台同时塞 LED 主屏+大舞台+大洽谈区（全挤一起都做不好）。

## 三、材质体系：材质有家族关系（一个展台最多三种主材）

| 调性方向 | 主材质 | 辅材质 | 点缀材质 | 灯光色温 |
|---|---|---|---|---|
| 黑色科技 | 深灰/黑色金属烤漆 | 亚克力/玻璃 | 蓝色灯带 | 冷白 5000-6000K |
| 白色极简 | 纯白烤漆面板 | 浅灰亚克力 | 金属银色线条 | 自然白 4000-5000K |
| 通透融合 | 玻璃/半透明材质 | 白色结构体 | 暖色灯光 | 中性白 4000K |
| 沉浸暗场 | 黑色哑光面板 | 深灰金属 | 霓虹色灯带 | 冷白+彩色重点光 |
| 高端商务 | 木饰面/皮革 | 香槟金属 | 暖光灯带 | 暖白 3000-3500K |
| 工业科技 | 裸露金属桁架 | 深灰面板 | 蓝/红灯带 | 冷白 5000K |
| 清新自然 | 木色/浅色 | 白色 | 绿植 | 暖白 3500-4000K |

规则：主材覆盖 60%+ 可见面积；辅材用于功能区分与层次；点缀材用于品牌识别与引导；**同一功能区材质 ≤2 种**；**相邻功能区材质要有过渡，不能突然跳变**。

## 四、灯光策略：三层，每层解决一个问题

| 层级 | 解决什么 | 典型形式 | 覆盖 |
|---|---|---|---|
| 基础照明 | 整个展台看得清 | 顶部嵌入筒灯、线性灯 | 全展台均匀 |
| 重点照明 | 引导视线到产品/品牌 | 射灯、投光灯 | 产品展柜、品牌墙、核心展品 |
| 氛围照明 | 营造空间感与情绪 | LED 灯带、灯箱、渐变光 | 结构边缘、地面边线、装饰面 |

与调性关系：科技感→基础压暗+重点强对比+冷色氛围；商务温馨→基础柔和均匀+暖色重点+暖色氛围；沉浸式→基础极暗+重点极亮+彩色氛围。
禁忌：❌ 全场均匀亮（像办公室）；❌ 只有氛围灯没重点灯（好看但产品看不清）；❌ 灯带色与品牌色冲突。

## 五、色彩体系：三色原则

| 角色 | 面积占比 | 作用 |
|---|---|---|
| 主色 | 60-70% | 定调，第一印象（通常品牌色或背景色） |
| 辅色 | 20-30% | 支撑，功能区分与层次（中性色或品牌辅色） |
| 点缀色 | 5-10% | 引导，吸引注意、品牌标识（品牌强调色或灯光色） |

与品牌关系：强品牌色（华为红、索尼黑）作主色或点缀，不能忽略；品牌色彩感弱时以调性方向为准、品牌色只作点缀；无论如何品牌 logo 颜色必须与展台色彩体系兼容。
